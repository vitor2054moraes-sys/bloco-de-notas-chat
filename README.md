Bloco de notas + Chat

Um aplicativo web **simples e profissional** para gerenciar tarefas, fazer anotações
e interagir por comandos — tudo rodando no navegador, sem servidor.

![Status](https://img.shields.io/badge/status-ativo-success)
![Licença](https://img.shields.io/badge/licença-MIT-blue)

## ✨ Funcionalidades

- ✅ **Lista de tarefas** com prioridades (🟢 baixa / 🟡 média / 🔴 alta)
- 📝 **Anotações dentro de cada tarefa** (expansíveis)
- 🗒️ **Bloco de notas independente** com formatação (negrito, itálico, sublinhado, listas)
- 💬 **Chat de comandos** para gerenciar a lista rapidamente
- 🎲 **Embaralhar** tarefas e ↩️ **voltar** à ordem original
- 🔍 **Filtros**: todas / ativas / concluídas
- 🧹 Remover concluídas com um clique
- 🌙 **Modo escuro/claro**
- 💾 Salva tudo automaticamente no navegador (LocalStorage)

## 💬 Comandos do chat

| Comando        | Ação                            |
|----------------|---------------------------------|
| `add tarefa`   | Adiciona uma nova tarefa        |
| `done nº`      | Marca a tarefa como concluída   |
| `del nº`       | Remove a tarefa                 |
| `clear`        | Apaga todas as tarefas          |
| `help`         | Mostra os comandos disponíveis  |
